BUDGET PHONE INTELLIGENCE
==========================

A clustering-based phone segmentation project that analyzes budget phones
scraped from Flipkart. The goal is to segment phones into meaningful groups
using KMeans clustering so buyers can quickly find the best phone for their needs.


WHY THIS PROJECT?
------------------
The Indian budget phone market is overwhelming — hundreds of models, varying specs,
and no clear way to compare them. This project:
- Scrapes real phone data from Flipkart
- Cleans and engineers useful features
- Segments phones into clusters like "Value King", "Budget Workhorse", and "Camera Champion"
- Provides an interactive Streamlit app to explore and filter phones


PIPELINE
--------
Step 1 | 01_null_analysis.py
  - Missing value audit
  - Mechanism detection (MCAR/MAR/MNAR)
  - KNN imputation for missing values

Step 2 | 02_eda.py
  - Univariate, bivariate, multivariate analysis
  - Generates plots for distributions, correlations, brand analysis

Step 3 | 03_feature_engineering.py
  - Creates new features: Value_Score, Camera_Score, Price_per_GB_RAM
  - Correlation-based feature selection (simple, no heavy ML)

Step 4 | 05_clustering.py
  - KMeans clustering with elbow/silhouette analysis
  - PCA for 2D visualization
  - Automatic cluster labeling based on centroids

Note: Step 4 (04_regression_models.py) was removed because this project
focuses purely on clustering, not rating prediction.


HOW TO RUN
----------
1. Install dependencies:
   pip install -r requirements.txt

2. Run the pipeline in order:
   python 01_null_analysis.py
   python 02_eda.py
   python 03_feature_engineering.py
   python 05_clustering.py

3. Launch the Streamlit app:
   streamlit run app.py


STREAMLIT APP PAGES
-------------------
Page 1 - Dataset Overview
  Browse and filter the full phone dataset by brand, price, 5G support.

Page 2 - EDA Dashboard
  Interactive charts: rating distribution, price vs rating, brand analysis,
  correlation heatmap.

Page 3 - Phone Recommender
  Set your budget, minimum RAM, battery preferences and get the top
  recommended phones sorted by value score.

Page 4 - Cluster Explorer
  View phone clusters on a scatter plot, pick a cluster and see its phones
  with average price and rating stats.


DATASET
-------
Source:       Flipkart (scraped)
Size:         1,532 phones, 26 brands
Price Range:  Rs.5,699 — Rs.69,999
Features:     Product Name, Brand, Price, Rating, RAM, ROM, Screen Size,
              Rear Camera, Front Camera, Battery, 5G support


CLUSTERS
--------
Value King
  High RAM + battery at low price — best bang for buck.

Budget Workhorse
  Affordable phones with decent specs.

Camera Champion
  Premium phones focused on camera quality.
